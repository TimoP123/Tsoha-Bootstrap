## Työn aihe

Reseptitietokanta on web-sovellus, jonka käyttäjät voivat jakaa ruokareseptejä keskenään. Sovelluksella voi myös hakea reseptejä raaka-aineiden tai reseptiin liitettyjen tägien avulla.
